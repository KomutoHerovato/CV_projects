# Проекты

В этой папке лежат проекты, которые удобнее показывать отдельно от учебных
материалов верхнего уровня репозитория.

## Разделы

- [ml](./ml) — ML pet-проекты, оформленные как самостоятельные мини-репозитории:
  код в `src/`, тесты в `tests/`, исходные ноутбуки в `notebooks/`, команды
  запуска и описание результатов.
