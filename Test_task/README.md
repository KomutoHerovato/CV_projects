# Сервис анализа успеваемости студентов

Тестовое задание: Разработка REST API на FastAPI для загрузки и анализа оценок.
Проект полностью асинхронный, упакован в Docker и работает с PostgreSQL.

## стек

*   **Python 3.11**
*   **FastAPI**
*   **PostgreSQL**
*   **asyncpg**
*   **Pydantic**
*   **Docker & Docker Compose** 

## Как запустить

Вам не нужно устанавливать Python или PostgreSQL локально. Достаточно иметь **Docker Desktop**.

1. **Клонируйте репозиторий** :
   ```bash
   git clone <ссылка-на-ваш-репозиторий>
   cd <папка-проекта>
   ```
2. **Запустите проект**
    ```bash
    docker-compose up --build
    ```
3. **Работа с запущенным проектом**
    http://localhost:8000
    перейдя по ссылке черзе пост, сверху добавьте .csv файл после чего в POST try it out -> execute
    аналогично пробейте методы get, чтобы сервис отобрал студентов
